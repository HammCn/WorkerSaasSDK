# 上报签约并派单

> [!IMPORTANT]
> 此服务为合作伙伴侧已明确完成了实名认证和签约，才可调用此接口。其他合作伙伴无法调用此接口。详细请咨询对接的BD老师。

## 接口地址

> task/push

## 请求参数

| 参数名称     | 必选 | 类型     | 描述   |
|----------|----|--------|------|
| taskId   | 是  | Long   | 任务ID |
| realName | 是  | String | 姓名   |
| idCard   | 是  | String | 身份证号 |
| phone    | 是  | String | 手机号  |

## 响应信息

如返回 200 状态码，则表示报名和签约成功，其他请参考返回错误信息。
