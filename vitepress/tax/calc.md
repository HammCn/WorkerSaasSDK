# 税额预算

> [!Warning]
> 税额预算仅供参考，最终税额将以实际发放为准。

## 接口地址

> tax/calc

## 请求参数

| 参数名称            | 是否必填 | 类型      | 描述                  |
|-----------------|------|---------|---------------------|
| idCard          | 是    | String  | 身份证号                | 
| balance         | 是    | Double  | 本次收入                |
| isEnterpriseTax | 否    | Boolean | 是否企业承担个人所得税(即算价税合计) |

## 响应参数

| 参数名称          | 类型     | 描述         |
|---------------|--------|------------|
| tax           | Double | 个人所得税      |
| vatTax        | Double | 将产生的增值税额   |
| additionalTax | Double | 将产生的附加所得税额 |
