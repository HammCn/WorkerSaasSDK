# 查询支付订单

> [!Tip]
> 除了我们的异步支付结果通知之外，你也可以通过此接口主动查询订单的支付结果。

## 接口地址

> order/query

## 请求参数

| 参数名称         | 是否必须 | 类型     | 描述                  |
|--------------|------|--------|---------------------|
| outerTradeNo | true | String | 外部订单号(您系统的订单号，保持唯一) |

## 响应参数

返回订单模型，详情请查看 [订单模型](../model/order.md)
